Privacy & Security Details

Data Flow

This application is a static HTML page with client-side JavaScript, running entirely within your browser sandbox. There is no backend server, no database, and no analytics. All form data is stored in localStorage on your device and can be cleared at any time by clearing your browser data.

What happens with your data

  • Auto-save: Responses are saved to your browser's localStorage (never transmitted)
  • Save Draft: Exports a JSON file to your local filesystem
  • Generate Document: Creates a self-contained HTML file downloaded to your machine
  • Print to PDF: Uses your browser's native print dialog
  • Uploaded files: Converted to base64 data URIs and stored in-memory; never uploaded to any server

External Dependencies

  • Google Fonts (fonts.googleapis.com) — loads DM Sans, Plus Jakarta Sans, and JetBrains Mono typefaces. This is the only external network request. No data is sent to Google; only font files are fetched.
  • No other external libraries, CDNs, or third-party scripts are loaded.

Source Code & Auditing

The complete source code is available for inspection at github.com/find-arka/tech-account-summary. You can verify that there are no fetch(), XMLHttpRequest, or any other network calls that transmit form data.
